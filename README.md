```shell
Go CLI application template for Go projects.

Usage:
  go-cli [command]

Available Commands:
  help        Help about any command
  version     Print the version/build number

Flags:
  -h, --help   help for go-cli

Use "go-cli [command] --help" for more information about a command.
```
